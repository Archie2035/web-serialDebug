##serial-webDebug项目
--------

![效果](./2016-03-20 23:42:33屏幕截图.png)

##简介
--------
对于web爱好者来说，在web上做一些事情真的很酷。这个项目通过使用serialport包调用底层的串口，完成串口的收发。和桌面软件的串口调试助手功能类似。
由于本人也是web新手，所以界面粗糙，代码组织的不够好。但是可以供大家参考和日常的使用。

##技术说明
--------
前端使用ejs模板引擎开发，通过jquery发起ajax请求与后端的nodejs进行交互。
